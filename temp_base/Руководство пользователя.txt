Версия программы: 1.0 alpha
Разработчик: Сапожников Владислав Валерьевич
Правила распространения: Бесплатный продукт с открытым кодом
Платформа размещения открытого кода: GitHub (gossipauthorxpm->temp_base->alpha-version)

При открытии программы пользователь видит окно ввода пользователя. При вводе ФИО абонента теплосети, имеющиеся в БД(прилогается SQL в проете на GitHub), производится переход
на страницу абонента.
Также на окне авторизации имеется кнопка панели админа. Логин и пароль от панели админа 1(логин), 1(пороль)
В этом окне возможно просмотреть всех абонентов теплосети. При просмотре открывается таблица, из которой можно копировать
данные нажатием на ячейку абонента.
Также можно просматреть данные отдельного абонента.
ФУНКЦИЯ "Отправить данные в РЭП" пока не доступна для использования.
